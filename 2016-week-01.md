### Android

* Android.mk에서 filter 사용 시 주의점

잘못된 예:

```
ifneq (($filter $(MY_TARGET), ver01 ver02), )
LOCAL_STATIC_JAVA_LIBRARIES := my.lib
endif
```

수정: 

**변수는 $로 시작한다.**

```
ifneq ($(filter $(MY_TARGET), ver01 ver02), )
LOCAL_STATIC_JAVA_LIBRARIES := my.lib
endif
```
